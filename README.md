# Evolution of the JS toolchain

Modern type-/javascript development is complex, utilizing a myriad of interconnected tools.

The presentation aims to capture (an incomplete and mostly inaccurate) history of 
Web tooling.

From the humble beginnings of static html deployed over FTP,
through the rise of ajax libraries and the jslint,
the framework wars and the explosion of tooling,
to the meta-frameworks and the dominance of vite.
